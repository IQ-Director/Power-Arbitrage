# v2 — CNN+BiLSTM 端到端

## 概述

在 v1 基础上大幅扩展：CNN 从原始 NWP 网格提取空间特征，分区统计捕获空间分布，修正预报补偿系统偏差，Trend-Aware Loss 直接优化电价趋势。

## 特征（367 维）

- 边界条件预测值 7 维
- 边界条件滞后 D-1 7 维
- NWP 全局统计 42 维（7 通道 × 6 统计量）
- NWP 分区均值 112 维（7 通道 × 16 子区域）
- NWP 衍生 5 维（风速、有效辐照度）
- 修正预报 42 维（3 种偏差估计 × 7 变量 × 2 类特征）
- 时间特征 12 维
- 价格滞后 4 维
- 价格滚动 8 维
- CNN 嵌入 128 维（从 7×104×225 原始网格提取）

## 模型（~3.8M 参数）

```
表格特征 (B, 96, 239) + CNN 嵌入 (B, 96, 128) → (B, 96, 367)
  → Linear(367→128) → LayerNorm
  → Conv1d(128→256, k5) → Conv1d(256→256, k5) → ReLU → Dropout
  → 2×BiLSTM(256, bidir) → Residual(128→512)
  → LayerNorm → Linear(512→1) → (B, 96)
```

CNN 编码器：3 阶段 ResBlock(32) + 下采样 → FC→128 维嵌入。

## Loss: Trend-Aware

```
Loss = 0.3 × 方向一致性 + 0.5 × Top-K窗口重叠 + 0.2 × Huber
```

## 训练

- Batch: 8（CNN 模式显存较大）
- 其余同 v1

| 项目 | v1 | v2 |

| 特征维度 | 85 | 367 |

| 修正预报 | 无 | 3 种偏差修正 |
| 模型参数 | ~830K | ~3.8M |
